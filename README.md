# BashOps Radar

A private startup opportunity intelligence tool built to accelerate Proof-of-Work client acquisition.

## What it does

BashOps Radar helps analyze startup GitHub repositories and identify:

- Open issues
- Recent activity
- Tech stack
- Opportunity score
- Best Proof-of-Work angle
- Suggested founder outreach message

## Goal

Turn manual startup inspection from 60–90 minutes into 5–10 minutes.
